Professional Techniques: The "Crop Duster" vs "Zoomer"

üé¨ The "Crop Duster" Method

  • Step 1: Generate ONE perfect wide shot
  • Step 2: Crop to medium, close-up, extreme close-up
  • Step 3: Animate each cropped version
  • Step 4: Edit together for film coverage

Pro: Guaranteed consistency
Con: Manual work required

üîç The "Zoomer" Method

  • Step 1: Generate close-up shot in Midjourney
  • Step 2: Use "Zoom Out" to create wider versions
  • Step 3: Animate each zoom level
  • Step 4: Professional film coverage achieved

Pro: Fast AI-powered workflow
Con: Less control over background

  • Physical traits: Size, colors, distinctive features

  • Expressions: Happy, sneaky, surprised, victorious

Character Prompting Best Practices

üéØ Consistency Prompting Formula

[Character Name] + [Key Physical Features] + [Action/Scene] + [Style] + [Mood/Lighting] Example: "Anansi the spider, eight legs, brown and gold coloring, traditional West African patterns, standing confidently in tropical forest, speaking to curious vervet monkeys, animated storytelling style, warm golden hour lighting, children's book illustration"

üí° Pro Tips for Better Results

  • Use the same opening: Always start with character name and key features
  • Include reference numbers: "Character sheet reference #1" in prompts
  • Consistent style tags: Keep art style terms identical
  • Cultural accuracy: Research authentic visual elements

Troubleshooting Common Issues

üé≠ Character Changes Face

Solution: Use more specific facial feature descriptions, try Leonardo's Character Reference

üé® Style Inconsistency

Solution: Keep exact same style keywords, use reference images, maintain consistent aspect ratios

üîÑ Background Changes

Solution: Use "Crop Duster" method, generate backgrounds separately, composite manually

‚ö° Motion Issues

Solution: Use motion brush selectively, start with subtle movements, avoid complex actions
